# getActivities( integer|null $iWeek = null, integer|null $iMonth = null, integer|null $iYear = null )

Returns all activities.

Returns all activities.

If - $iWeek is set, only the requested week will be returned. - $iMonth is set, only the requested month will be returned. - $iYear is set, only the requested year will be returned.

$iWeek and $iMonth can be used together with $iYear. if $iYear is null, the current year will be used for filtering.

Parameters

Number of the wanted week.
$iMonth
Number of the requested month.
